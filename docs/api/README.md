# API Reference

This section provides detailed documentation for all public packages and modules in the syn library.

## Packages

### [app](./app.md)
Core client library for interacting with Synthetic.new API, including chat, embeddings, vision, and model listing capabilities.

### [cmd](./cmd.md)
CLI commands and application entry point for the syn tool.

### [config](./config.md)
Configuration management and default values for the application.
